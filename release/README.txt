AutoClicker release package

Files
- AutoClicker.exe: main executable

How to use
1. Run AutoClicker.exe on Windows.
2. Add mouse or keyboard actions.
3. Set start delay if the target app is full-screen.
4. Click Start, then switch back to the target app.

Tips
- If the target app is running as administrator, run this tool as administrator too.
- Configuration is saved to %AppData%\AutoClicker\config.json.
